Before every task: scan your available skills list. If any skill matches what you're about to do, invoke it with the Skill tool before proceeding.

**Rules:**
- Skills are mandatory. A matching skill must be used.
- Announce before invoking: "I'm using [skill] to [purpose]."
- If a skill has a checklist, create a TaskCreate todo for each item.

**Reject these rationalizations:** "This is simple" / "I remember the skill" / "The skill is overkill" — none are valid. Check the list; use the skill.
